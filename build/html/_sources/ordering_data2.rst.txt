Ordeting Data
=============

Ordering data from the EUMETSAT archive
---------------------------------------

1. Log in to
   `https://eoportal.eumetsat.int <https://eoportal.eumetsat.int/>`__
   with:

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_1.png
   :alt: Image 1

   Image 1
::

    username: hdeneke
    password: S3V1R1umarf

2. Start the data centre application using **Access** under **Data
   Centre**.

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_2.png
   :alt: Image 2

   Image 2

3. Select your data type from the data center product list, then go on
   with **next step**:

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_3.png
   :alt: Image 3

   Image 3
- For **Prime-Service** (PZS) use: *"High Rate SEVIRI Level 1.5 Image
Data -- MSG -- 0 degree"*

- For **Rapid-Scan-Service** (RSS) use: *"Rapid Scan High Rate SEVIRI
Level 1.5 Image Data -- MSG"*

- For **Prime-Service over Indian Ocean** use: *"High Rate SEVIRI Level
1.5 Image Data -- MSG -- Indian Ocean 41.5 degrees E"*

4. Double-check if the required longitude in **Sub Sat Longitude** is
   correct:

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_4.png
   :alt: Image 4

   Image 4
- 0° for the Prime-Service (PZS)

- 9.5° for the Rapid-Scan-Service (RSS)

- 41.5° for the Indian Ocean

5. Select the required period of time in the **Select Date / Time**
   field. Get results with **Apply** (might take a while).

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_5.png
   :alt: Image 5

   Image 5
- a maximum of **three months** for Prime-Service (PZS)

- a maximum of **one month** for Rapid-Scan-Service (RSS)

6. Skip the selection of regions using **Next Step**.

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_6.png
   :alt: Image 6

   Image 6

7. Select product order format as "**HRIT data sets in tar file**\ ".
   Then click next step.

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_7.png
   :alt: Image 7

   Image 7

8. Go to **Details**, which on the right to **next step**.

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_8.png
   :alt: Image 8

   Image 8
Use ◄ and ► to browse through the selected period of time and use **CTRL
+ Left Click** to select the slots you want. If you are done with the
selection use **Remove Unselected** to clear all unwanted slots from
your list. Double-check if no slot is missing. Use "Close" if everything
is correct.

9. Choose the **Delivery method** which is suitable for the data:

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_9.png
   :alt: Image 9

   Image 9
→ "Online HTTP" if the order is ≤80 GB

→ "On Media" if the order is >80 GB

By delivery option **no compression** is fine because the data files are
already compressed.

10. Go the last step, where only the overall data size is presented.

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/_static/ordering_data_10.png
   :alt: Image 10

   Image 10
Finally send the order with "Place your order".

11. Note the order number at our wiki:

http://wiki-intern.tropos.de/index.php/EUMETSAT_Data_Ordering_Diary

.. figure:: /vols/satellite/home/bayer/Satelite_gaps/source/images/ordering_data_11.png
   :alt: Image 11

   Image 11

12. Click at **ORDER STATUS**. It shows the order number and the status
    of the order. Several status are possible:

- Pending (order still not submitted), - Submitted (order is submitted),
- Cancelled (order is cancelled), - Processing (order is en route), -
Delivered (order is delivered), - Error (order went wrong)
